e512 - HTML - iframes

1.
 Using internal frames

With HTML you can use internal frames. Internal frames act like a browser window inside the current window. They can scroll and display HTML pages just as a normal borwser window does. It's simply the chance to build a more modulare web page which different html feeds into one. Here is an example:

 
 

Show e501 inside iframe

2.
 Code

<html>
 <head>
 </head>
 <body>
   <iframe src="http://www.google.com" width="300" height="400" name="winbox">
     <p>Your browser does not support iframes</p>
   </iframe>
 </body>
</html>
 

The above code embeds an internal frame into the current page. Clicking on links inside the frame will display the content again inside of it. Here is a bit more complex example:

 
 

...
<iframe src="http://www.google.com" width="300" height="400" name="winbox" scrolling="yes" marginheight="0" marginwidth="0" frameborder="0">
     <p>Your browser does not support iframes</p>
</iframe>

<a href="http://www.yahoo.com" target="winbox">yahoo</a>
...

 

iframes accept a range of attributes all well known from different tags such as the window (make scrollbars visible or not) or the frameborder, marginswith, etc.

Important is the "name" attribute since this allows to direct anchors from one document to display inside the iframe. In the above example, yahoo will be shown inside the inframe given by as target.

*** Stage ***

 

Two layers both with the same width of 600px. They are centered by margin-right:auto; and margin-left:auto;

Note that the first CSS sets all margins and paddings to 0. Otherwise there would be some default spacing between the layers!

Also, no height for the Stage is specified. This means it grows with the content.

 

* {
    padding: 0;
    margin: 0;
}

div#banner{
    width:600px;
    height:50px;
    margin:40px auto 0px auto;
    background-color:#FFCC99;
}

div#Stage{
    width:600px;
    margin:0px auto 0px auto;
    background-color:#6699FF;
}

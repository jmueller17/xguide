e534 - CSS - Float



*** layer01 ****

position:relative;
float:left;

Hello this is a text supposed to flow around the layer. Depending on the two values "float:left | right " the layer will either stay on the left side or jump to the right border in relation to the text. Be sure to check out the source code to see that the code sequence does not correspond to the visible order. The layer code comes in both cases first, followed by the text.

*** layer02 ****

Hello this is the same text but now in relation to the second layer. Depending on the two values "float:left | right " the layer will either stay on the left side or jump to the right border in relation to the text. Be sure to check out the source code to see that the code sequence does not correspond to the visible order. The layer code comes in both cases first, followed by the text.

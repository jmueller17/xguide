e541 - CSS - Image Rollover
 

You can code an image rollover with pure CSS simply by swapping the background image linked to the pseudo-format "a:hover".

a.swapped {
    display:block;
    width:100px;
    line-height:100px;
    text-decoration:none;
    background-image: url(path/to/image1.gif);
    background-repeat:no-repeat;
}

a.swapped:hover {
    background-image:url(path/to/image2.gif);
    background-repeat:no-repeat;
}

 

Note that the width and line-height values actually correspond to the dimensions of the image. The actual link in the HTML document is coded like that:

<a href="#" class="swapped">&nbsp;</a>

This would be some random text. Here we have the default link. No special styles attached with the "class" selector.

But of course we can also define in the CSS a first custom anchor style which is in action here. Note that you link this custom style by simply introducing the "class" selector in the anchor tag: <a class="style1" ...></a>

Then, to make the case, I second custom style for yet another anchor. See it in action on this very link. Same thing as above, but now you just change the attached style: <a class="style2"...></a>

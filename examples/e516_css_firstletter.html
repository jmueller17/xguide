This is an example of how to make the first letter big with the pseudo-format "first-letter". Check it out in the source code. You'll see that the text starts right away in the <p> paragraph tag. Essential is the "float:left" value which moves the subsequent text nicely around the initial letter. Put in some margins and you are done.

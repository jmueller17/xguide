Similar to the e517 a hover effect can include a background image. Here the anchor works with the well known pseudoformat "a:hover". Both, however also include background-image:

a {
    padding-left:20px;
    margin-bottom:8px;
    margin-top:8px;
    background-image: url(../img/bullet2.gif);
    background-repeat:no-repeat;
}

a:hover {
    background-image:url(../img/bullet.gif);
    background-repeat:no-repeat;
}

 

Note also the "padding" which pushes the text actually to the right in order to avoid overlaying on the image and the "margin" which introduces the spacing between the items.

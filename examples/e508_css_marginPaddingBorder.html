CSS Padding - Margin - Border

Padding and margin allows you to control the spacing between xHTML elements. To style the sides of elements, use border.

border

The border style of any xHTML element is controled by three properties:

a) border-color: any color value
b) border-width: thin | thick | medium | numeric value
c) border-style: dotted | dashed | solid | double | groove | ridge | inset | outset

Each of the four borders can be controlled individually with border-left, border-top, border-bottom, and border-right. One can also specify all four sides at the same time with border:color width style

<p style="border:#0033FF 3px dotted;">blue 3px dotted border</p>
<p style="border-left:#6600CC 8px solid;">just the left<br/> border</p>
<p style="border-top:#FF6600 1px ridge;">just the top border</p>

margin

margin: defines the space between a selected element and its neighbouring, external elements. If you want to create space around an element use margin. You can either select each of the four sides individually or specify all at once.

You can specify each side invididually with margin-top, margin-right, margin-bottom, margin-left. Or all in one: margin: top right bottom left. The sequence is important!

<html>
 <head>
   <title>e508 - CSS - Margin</title>
   <style type="text/css">
        p {
            margin-top:40px;            
            border:#000099 1px solid;
         }
   </style>
 </head>
 <body>
        <p>First paragraph with margin-top:40px</p>
        <p>Second paragraph with margin-top:40px</p>
 </body>
</html>

Show | Open

What is important in the above example is to note how margin introduces spacing between the paragraphs.

padding

padding: defines the spacing between a selected element and its content. If you want to avoid text sticking to the border of a layer for example, use padding. Similar to margin you can set each side individually or all in one with padding: top right bottom left.

<html>
 <head>
   <title>e508 - CSS - Padding </title>
   <style type="text/css">
        p {
            padding:100px 40px 1px 30%;            
            border:#000099 1px solid;
         }
   </style>
 </head>
 <body>
        <p>First paragraph with padding set </p>
        <p>Second paragraph with padding set </p>
 </body>
</html>

Show | Open

Note how padding introduces a spacing between the border of the paragraph (blue line) and its content. To spell it out:

padding-top: is set to 100px
padding-right: to 40px
padding-bottom: 1px
padding-left: to 30% of the overall document width.

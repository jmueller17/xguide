 Basic Table Structure

<table>
 <tr>
  <td>1</td>
 </tr>
</table>

Basic table structure, one row one cell. Row encloses cells (columns)

<tr> indicates table row

<td> table cell

  <td>a1</td>
  <td>a2</td>
 </tr>
</table>

<table width="20">

Table with fixed width of 20 pixels
  <table width="80%" align="center" >
  <tr>
    <td>1</td>
  </tr>
</table>

Table with relative width of 80% of the browser window. Resizing the browser window will resize the table!

This table is also centered in the browser window.

  <table>

 </tr>

a1
b1 b2

Table containing two rows. The first row spans two colums, therefore the attribute colspan="2"
  <table>
 <tr>
  <td rowspan="2">a1</td>
  <td>a2</td>
 </tr>
 <tr>
  <td>b2</td>
 </tr>
</table>

 

a1 a2
b2
Now, we use rowspan="2" which runs one cell over two rows.

2.
 Some Tips
  • Using tables with an invisible border is a very effective and "cheap" layout tool

  • Using emtpy cells only containing a transparent 1-pixel gif as a placeholder scaled to the required dimensions with the width/height attribute of the <img> tag produces very precise results in creating margins.

  • Nesting tables (a table inside a table) is in-efficient in page parsing and time needed to display the whole page. Avoid it where possible.

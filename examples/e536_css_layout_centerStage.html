*** Stage ***


A very simple layout. The blue layer is centered through the "margin" attribute set on "auto". It has a fixed width and fixed margins from the top. "Padding" introduces some space between the text and the border:

div#Stage {
  margin:40px auto;
  padding:40px 20px 40px 20px;
  width:750px;
  height:500px;

  background-color:#0066FF;
  font-size:18px;
  font-weight:bold;
}

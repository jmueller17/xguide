1.
 Position

<html>
 <head>
 <style>
 #s1 { position:absolute;
       top:180px;
       left:20px;
       width:80px;
       height:50px;
       background-color:blue;
    }
 </style>
 </head>
 <body>
   <div id="s1"> ...</div>
 </body>
</html>

 

Absolute positon means the location of the element can be specified in absolute pixel values in relation to the top-left or bottom-right corner of the page. The code produces the blue square on the left. Resize the window and look what happens!

    }

2.
 Stacking or z-index
 

<html>
 <head>
 <style>
 #s3{ position:absolute;
      top:600px;
      left:10px;
      z-index:1;
      width:60px;
      height:60px;
      background-color:#99ff00;
    }
 #s4{ position:absolute;
      top:620px;
      left:20px;
      z-index:50;
      width:65px;
      height:60px;
      background-color:#999966;
    }
 </style>
 </head>
 <body>
   <div id="s3">z-index:1</div>
   <div id="s4">z-index:50</div>
 </body>
</html>

The z-index determines the stacking of the layers. Lower values mean underneath higher values.
